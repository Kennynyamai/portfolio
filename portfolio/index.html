Pal Distributed Version Control System

Pal DVCS, a custom distributed version control system inspired by Git, with unique commands to avoid conflicts with standard Git software. It supports staging, committing, branching, diffs, merging, cloning, and .palignore. Packaged as an npm module for easy installation and use, this project deepened my understanding of distributed version control systems.

Design and Fabrication of a Dual Recovery System for the N4 Rocket

Designed and implemented a dual parachute recovery system for the Nakuja Project as part of my final year Mechatronic Engineering project. The system deploys a drogue parachute at apogee and a main parachute at a lower altitude for controlled landing near the launch site. It monitors flight data to determine optimal deployment timing, altitude, and mechanisms. Collaborated with the recovery team on design, development, and implementation.
